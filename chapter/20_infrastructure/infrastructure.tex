\chapter{Infrastruktur}

\section{Server}
Auf dem Server laufen folgende Komponenten. \\
%\textbf{Team City: } Dient als Continuous Integration server, führt bei jedem commit alle Tests durch. Ausserdem sind auch Build jobs eingerichtet, um ein Deployment anzustossen.
%\textbf{RabbitMQ: } Der RabbitMQ Broker läuft auf demselben server, wurde mit der Management Console eingerichtet, um die aktiven Queues zu überwachen.
%\textbf{Helin Server: } Der aktive Helin Server. Dieser wird bei Bedarf mit dem neusten Code deployt.
